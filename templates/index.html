Water Quality Prediction – Interactive Dashboard

Machine learning–based visualization of river water pollutant levels

About This Project

This is a personal machine learning project focused on predicting river water quality parameters using historical monitoring data and interactive visual analytics.

  • Dataset: Southern Bug River Water Quality (Kaggle)
  • Model: Random Forest Regressor (Multi-Output)
  • Notebook: Training & Analysis
  • Parameters Description PDF: Parameters PDF
  • Base GitHub: Base Project Repository
  • Deployed GitHub: Deployed Project Repository

⚠ Disclaimer: This is a personal project. Predictions are indicative and must not replace certified laboratory testing.
